# Jellyfin Music Replay

Generate music replay playlists for Jellyfin, based on the database generated by the [Playback Reporting](https://github.com/jellyfin/jellyfin-plugin-playbackreporting) plugin.

## Compatibility

- Jellyfin Server: `10.10.7` [openapi.json](https://api.jellyfin.org/openapi/stable/jellyfin-openapi-10.10.7.json)
