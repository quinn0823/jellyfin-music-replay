# Jellyfin Music Replay Playlist Generator

Jellyfin Music Replay Playlist Generator, based on the playback_reporting.db generated by the [Playback Reporting](https://github.com/jellyfin/jellyfin-plugin-playbackreporting) plugin.

## Compatibility

- Jellyfin Server: `10.10.7` [openapi.json](https://api.jellyfin.org/openapi/stable/jellyfin-openapi-10.10.7.json)
